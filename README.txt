Utilisation : java -jar ChatServer.jar [ OPTION ]...
-a , -- address = ADDR spécifier l'adresse IP
-h , -- help afficher l ' aide et quitter
-m , -- multicast démarre le client en mode multicast
-n , -- nio configurer le serveur OU le client en mode NIO
-p , -- port = PORT spécifier le port
-s , -- server démarrer le server
